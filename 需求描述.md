# 需求描述
我期望实现一个可代理及可编排的java服务；支持自动代理到下游服务，也可编码方式调用下游服务，spring生态

## 需求清单
### studio-demo
* 路由功能
    * 支持特定域名/IP地址路径的动态路由
    * 支持按特定接口维护的路由
    * 支持重写返回报文体结构
    * 支持打印请求和返回报文并统计耗时时间
* 编排功能
    * 支持编码方式调用下游服务

### mvc-demo
* 作为studio-demo的下游服务，提供rest接口，提供2个接口
    * /mvc/demo/hello 返回 yy
    * /mvc/demo/hello/{name} 返回name


返回结构
```json
{
    "code":"xxx",
    "msg":"xxx",
    "data":{}
}
```
